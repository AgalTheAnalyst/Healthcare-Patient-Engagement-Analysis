# Data Source
The datasets used in this project came from CMS Open Data Portal.
HCAHPS Patient Experience Dataset: https://data.cms.gov/provider/dataset/dgck-syfz

To reproduce this analysis:
1. Download the dataset from the CMS link above.
2. Save file as: hcahps_hospital.csv
3. Place it in the following directory: data/raw/
